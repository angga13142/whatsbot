## 3. Basic Features

### Recording Transactions

#### Record Income (Paket)

```
/paket [amount] [description]

Example:
/paket 1500000 Pengiriman paket ke Jakarta
```

**Alternative format:**

```
/catat paket 1500000 Pengiriman paket ke Jakarta
```

#### Record Expense (Jajan)

```
/jajan [amount] [description]

Example:
/jajan 250000 Bensin dan tol
```

#### Record Debt (Utang)

```
/utang [amount] [customer] [description]

Example:
/utang 500000 Toko ABC Pembelian barang
```

---

### Viewing Transactions

#### View History

```
/history

Or specify days:
/history 7    (last 7 days)
/history 30   (last 30 days)
```

#### View Today's Report

```
/laporan
```

Shows summary of today's transactions.

## 4. Transaction Management

### Transaction Status

- **Pending:** Awaiting approval (amounts > Rp 1,000,000)
- **Approved:** Verified and final
- **Rejected:** Not included in reports

### Quick Actions

**View pending transactions:**

```
/pending
```

**Check specific transaction:**

```
/detail [transaction-id]
```
