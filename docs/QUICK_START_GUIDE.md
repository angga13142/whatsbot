# WhatsApp Cashflow Bot - Quick Start Guide

**Get started in 5 minutes!**

---

## Step 1: Start the Bot (1 minute)

Send to the bot:

```
/start
```

You'll see a welcome message and your role.

---

## Step 2: Record Your First Transaction (2 minutes)

**For Income:**

```
/paket 500000 Pengiriman ke Jakarta
```

**For Expense:**

```
/jajan 50000 Makan siang
```

**For Debt:**

```
/utang 1000000 Toko ABC Pembelian barang
```

‚úÖ Done! Transaction recorded.

---

## Step 3: View Your Report (1 minute)

```
/laporan
```

You'll see:

- Today's income
- Today's expenses
- Net cashflow
- Transaction list

---

## Step 4: View Dashboard (1 minute)

```
/dashboard
```

Shows comprehensive overview with charts and insights.

## Common Commands

| Command                             | What It Does        |
| ----------------------------------- | ------------------- |
| `/paket [amount] [desc]`            | Record income       |
| `/jajan [amount] [desc]`            | Record expense      |
| `/utang [amount] [customer] [desc]` | Record debt         |
| `/laporan`                          | Today's summary     |
| `/history`                          | Transaction history |
| `/dashboard`                        | Visual dashboard    |
| `/help`                             | Show help menu      |
